# RTI (Real Time Infrastructure)

New infrastructure to perform small requests to our talent network that enable
real-time dispatching and matching.

This library will provide core support for each application that will use the
infrastructure

## Contents

This library will include:

- Reusable queries and mutations
- Core types
- Reusable components
- Utilities

## Usage

Currently there are two applications using library, `talents-gig-requests` and
`talents-gig-candidates`. There are plans to expand the gig space with multiple
usecases
