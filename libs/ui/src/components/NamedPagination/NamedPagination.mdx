## NamedPagination

Use `NamedPagination` to show a pagination component displaying a name for each page.

It provides back and next buttons when there are multiple pages.

  <Story id='ui-namedpagination--multiple-pages' />
</Canvas>

### One page

When there's only one page, the back and next buttons are not shown.
